# Terminal Hand Cricket: Ultimate Edition
* A fully playable, Python-based terminal game that brings the classic playground game of Hand Cricket to the digital screen. Designed with a custom local-multiplayer architecture, this game utilizes the pwinput library to mask keystrokes, ensuring fair 1v1 and Team battles on a single keyboard.

# Features:

* Three Game Modes: Play Solo against an RNG computer, Duo in 1v1 matchups, or Team Mode with custom squad sizes.

* Fair-Play Engine: Uses hidden password inputs for bowling and batting so players can't screen-peek.

* Dynamic Scoring & Logic: Tracks wickets, calculates target scores, and manages innings shifts automatically.
